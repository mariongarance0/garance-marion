# 🏎 F1 Data Lab

**Formula 1 analytics platform combining historical data, custom metrics, and predictive analysis.**

This project aggregates large-scale Formula 1 datasets and provides tools to explore race performance, driver comparisons, and race dynamics.

The platform is built around the open-source database:

* [https://github.com/f1db/f1db](https://github.com/f1db/f1db)

which contains decades of historical F1 data.

I extended this dataset with additional information, including **overtaking statistics**, enabling deeper analysis of race dynamics.

## Key Features

### 📊 Advanced F1 Data Analysis

The platform centralizes large amounts of racing data:

* drivers
* races
* teams
* lap results
* standings
* overtakes

These datasets allow detailed performance analysis across seasons.

### 📈 Custom Performance Metrics

I developed custom metrics to better analyze driver performance, including internal team comparisons and race dynamics indicators.

These metrics aim to provide insights beyond traditional statistics.

### 🤖 Machine Learning Analysis

Machine learning models are used to explore how different variables influence race outcomes, such as:

* grid position
* overtaking opportunities
* team performance
* circuit characteristics

The goal is to better understand **which factors most strongly impact race results**.

### 🎮 Interactive User Experience

Users can create accounts and interact with the platform through personalized features such as:

* race predictions
* F1 bingo challenges
* personalized statistics

This transforms the platform from a static dataset into an **interactive F1 analytics experience**.

# 📝 TagNote

**A structured note-taking system combining hierarchical folders and flexible tag-based search.**

Most mobile note-taking apps store notes in a simple list, which becomes difficult to navigate over time.

TagNote was designed to improve how users **organize, retrieve, and structure their information**.

🌐 Live app
[https://tagnote.lovable.app](https://tagnote.lovable.app)

## Core Concepts

TagNote combines two complementary organizational systems.

### 🗂 Folder Hierarchy

Notes can be organized using folders and subfolders, similar to a traditional computer file system.

This enables clear long-term organization.

### 🏷 Tag-Based Filtering

Users can create custom tags and attach them to notes.

Search queries allow filtering notes by:

* single tags
* combined tags
* exclusive tag filters

This makes it easier to retrieve information quickly.

### 📱 Multi-Device Access

Users can create accounts using their email in order to access their notes from multiple devices.

### ✅ Integrated To-Do Lists

The platform also includes simple task management features to track daily objectives.

# 🌙 Oniris – Dream Journal & AI Interpretation

**A social dream journal that tracks dreams over time and provides AI-based interpretations.**

🌐 Live app
[https://oniris-dream.vercel.app](https://oniris-dream.vercel.app)

Dreams are often forgotten quickly, and traditional note-taking tools are not optimized for dream journaling.

Oniris provides a dedicated platform to **record, analyze, and share dreams**.

## Key Features

### 🌙 Dream Tracking

Users can log their dreams and maintain a complete dream history.

Each dream entry contains:

* title
* description
* tagged people
* privacy settings

### 📊 Dream Statistics

The application generates statistics about dream activity, including:

* dreams per week
* dreams per month
* long-term tracking trends

### 👥 Social Interaction

Users can add friends and share dreams with them.

Dreams can be:

* private
* visible to all friends
* visible only to tagged users

### 🤖 AI Dream Interpretation

After writing a dream, users can request an AI analysis that attempts to interpret possible symbolic meanings and emotional themes.

These interpretations aim to stimulate reflection and curiosity about the subconscious mind.
